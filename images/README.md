# Images Folder

## Hero Background Images

Add your hero section background images here with these names:

- `hero-bg-1.jpg` (or .png, .webp)
- `hero-bg-2.jpg`
- `hero-bg-3.jpg`
- `hero-bg-4.jpg`
- `hero-bg-5.jpg`

**Recommended specifications:**

- Resolution: 1920x1080 or higher
- Format: JPG, PNG, or WebP
- File size: Optimize for web (under 500KB per image)
- Content: Athletic/training/sports-related imagery

The images will automatically rotate every 5 seconds on the home section background.

## Adding More or Fewer Images

If you want to use more or fewer images, update the `heroImages` array in `script.js` (around line 273).
